# Timed-Math-Challenge

A fun and fast-paced command-line math quiz built in **Python** that tests your arithmetic speed. Solve 10 randomly generated problems as quickly as you can — your time is being tracked!
 
---
 
## ✨ Features
 
- 🎲 Randomly generates math problems every round
- ➕➖✖️ Covers addition, subtraction, and multiplication
- ⏱️ Tracks total time taken to complete all problems
- 🔁 Keeps asking until you get the correct answer
- 📊 Counts wrong attempts across the session
- ⚙️ Easily configurable difficulty and problem count
---
 
## 🛠️ Tech Stack
 
- **Language:** Python 3
- **Libraries:** `random` (built-in), `time` (built-in) — no installs needed!
